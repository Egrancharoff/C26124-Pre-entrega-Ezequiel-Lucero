# C26124-Pre-entrega-Ezequiel-Lucero
-----------------------------------------------
Este proyecto es la elaboracion de un sistema de gestion de activos.

Se encuentra dividido en secciones:

Introduccion > Breve texto que explica el sistema.
Listado de vehiculos > Separado en cards para acceder a otras paginas que den el listado de vehiculos ordenados segun el tipo de equipo.
Portal de documentacion > Seccion para carga y visualizacion de documentos de los activos.
Actualizacion de base de datos > Seccion para acceder a los formularios para modificar las bases de datos.
Reportes > Seccion para acceder a la visualizacion en pantalla de las bases de datos.
Contacto > Seccion para enviar un formulario de contacto.
